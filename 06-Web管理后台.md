## 1. 本阶段目标

搭建 Web 管理后台，作为 Agent 和工具的"配置工厂"以及任务运行的"监控中心"。

**成功标准**：

- 可视化创建/编辑 Agent（选模型、配工具、写 System Prompt）
- 工具注册中心：浏览可用工具、配置 API Key、查看调用统计
- 任务模板库：预设常用任务，一键复用
- 实时监控台：运行中 Agent、执行日志、token 消耗、费用统计
- 知识库管理：上传文档、查看索引状态、测试检索
