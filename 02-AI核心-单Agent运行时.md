## 1. 本阶段目标

搭建 AgentOS 的"心脏"——让一个 AI Agent 能独立完成用户交给的任务。

**成功标准**：

- 给 Agent 一个任务（如"研究 AI Agent 最新论文并生成 500 字摘要"），它能自主调用工具完成
- 支持流式输出，用户可实时看到 Agent 的思考和执行过程
- 支持 ReAct 循环（Think → Act → Observe → Repeat），自动决定何时结束
