## 1. 本阶段目标

搭建 NestJS API 网关，作为移动端和 Web 后台的统一入口，负责认证、路由、流式中继。

**成功标准**：

- 用户注册/登录（JWT + OAuth）
- 完整的 RESTful API：任务 CRUD、Agent 配置、工具管理
- SSE 流式推送 Agent 思考和执行过程到前端
- Rate Limiting、输入校验、API Key 管理
