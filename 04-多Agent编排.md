## 1. 本阶段目标

实现 AgentOS 的核心差异化能力——多个专业化 Agent 协作完成复杂任务。

**成功标准**：

- 用户给一个复杂任务 → 系统自动拆成子任务 → 分配给合适的 Agent
- 支持 3 种协作模式：流水线（A→B→C）、并行分发、辩论模式
- 多个 Agent 的产出能自动合并、去重、格式化
